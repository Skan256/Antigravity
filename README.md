# Antigravity

Projet Next.js avec Firebase et Gemini AI.

## Prérequis

- [Node.js](https://nodejs.org) v18 ou plus
- Git

## Installation

**1. Cloner le repo**
```bash
git clone https://github.com/Skan256/Antigravity.git
cd Antigravity
```

**2. Installer les dépendances**
```bash
npm install
```

**3. Créer le fichier `.env.local` à la racine**

Demander le fichier `.env.local` à Skander et le placer à la racine du projet.

**4. Lancer le projet**
```bash
npm run dev
```

**5. Ouvrir dans le navigateur**

http://localhost:3000
